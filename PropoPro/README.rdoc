== Creative Proposal Generator - PropoPro


Installation

1. Go ahead and fork and clone this repository.
2. Then go and "bundle install" to make sure all the required Gems will function in this program.
3. "rake db:migrate"
4. Start Your Web Server - "rails s"
5. Start creating. 

I will be adding a feature which shoots out emails as new users are created, but that is still in the works.

Enjoy, and Happy Coding.
